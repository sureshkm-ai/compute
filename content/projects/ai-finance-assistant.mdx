## Overview

AI Finance Assistant is a conversational agent that lets you ask plain-English questions about your personal finances and get structured, grounded answers — no spreadsheets required.

## What it does

- **Natural-language queries** — ask "How much did I spend on food last month?" and get a direct answer
- **Spending summaries** — automatic categorisation and period-over-period comparisons
- **Insight surfacing** — the agent flags unusual transactions and highlights saving opportunities
- **Streamlit UI** — clean, browser-based interface deployable in one click

## How it works

The agent combines an LLM with Pandas-backed tool calls to query your financial data. A LangChain tool layer translates user intent into structured data operations, so the model never hallucinates figures — it always derives answers from the actual numbers.

## Try it

The app is live at [finance-agent-ai.streamlit.app](https://finance-agent-ai.streamlit.app/) — upload a CSV export from your bank and start asking questions.
